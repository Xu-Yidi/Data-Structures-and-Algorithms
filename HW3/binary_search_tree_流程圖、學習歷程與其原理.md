### Binary Search Tree
- **What's Binary Search Tree**
二元搜尋樹(Binary search tree, BST)是二元樹的一種，且其每個節點(node)的值(key)滿足下列特性：值比該節點的值小的節點均在該節點的左子樹，而值比該節點的值大的節點均在該節點的右子樹<br>
